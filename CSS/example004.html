Propiedades y Valores Básicos

Color y Fondo

CSS permite controlar el color de fondo de elementos HTML mediante la propiedad background-color. Se pueden utilizar valores como nombres de colores, códigos hexadecimales o valores RGB.
La propiedad color se utiliza para definir el color del texto dentro de un elemento.

Texto y Fuente

La propiedad font-family se usa para definir el tipo de fuente que se aplicará al texto.
La propiedad font-size controla el tamaño del texto en píxeles, puntos o ems.
Otras propiedades, como font-weight (grosor de la fuente) y font-style (estilo de la fuente), permiten personalizar aún más la apariencia del texto.
